# Seed Identifier Agent

Seed selection expert that recommends optimal seed varieties with real buying options worldwide.

## Purpose
Helps farmers identify and purchase quality seeds suited to their location with actual online buying links and prices.

## Tool
- `google_search`: Researches seed varieties, suppliers, prices, and real purchasing links

## Recommendations Include
- Top 3 seed varieties for location
- Seed properties (heat-tolerant, drought-resistant, disease-resistant)
- Expected yields and maturity periods
- Real online buying links with prices
- Government/official sources
- Quality certification checks
- Seed treatment recommendations
- Cost analysis per hectare

## Adapts To
- Local currency (USD, EUR, INR, etc.)
- Local measurement units (hectare/acre)
- Country-specific suppliers and regulations
- Regional agricultural systems

## Returns
- Seed variety recommendations with scientific backing
- Beautiful markdown cards with buying options
- Clickable links to verified suppliers
- Price comparisons
- Quality check guidelines
- Visual guides (seed appearance, packaging, mature plants)

## Usage
Answers questions like "Which seeds should I buy?" or "Best seed varieties for my location?"
