# Yield Improvement Agent

Optimization expert that suggests strategies to increase crop production and yields.

## Purpose
Provides scientifically-backed recommendations to maximize crop yields based on location and current practices.

## Tool
- `google_search`: Researches best practices, fertilization schedules, pest management, timing optimization

## Strategies Provided
- Optimal fertilization schedules (NPK timing and quantities)
- Pest and disease management
- Irrigation optimization
- Planting density and spacing
- Variety selection
- Soil health improvement
- Harvest timing
- Post-harvest handling

## Returns
- Specific improvement strategies with data
- Implementation timeline
- Expected yield increase percentages
- Cost-benefit analysis
- Visual guides for techniques

## Usage
Answers questions like "How to increase yield?" or "Best practices for higher production?"
